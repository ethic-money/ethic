# ∉thic
Ethical, decentralized self-stabilizing money. Not an element of fiat monetary systems.

## Stablecoin

- shorthand name: `∉thic`
- full name: `Ethic Money`
- symbol: `ETHC`
- decimals: `18`
- character: `∉ [U+2209]`
